This module provides user login redirect by roles.

*Introduction:
Some times your clients have wonderful ideas and one of this ideas is redirect the users to different pages depending of the roles what they have, in that case "User Login Redirect" is your simple and fast solution, we redirect the users depending of his roles and the weight of this roles.

*Installation:
Install as you would normally install a contributed drupal module.
See: https://drupal.org/documentation/install/modules-themes/modules-7 for further information.

*Configuration:
 - Go to admin/configure/user-login-redirect this route provides the configuration form for this module.
 - Specify one by one the redirect routes and they wieght by user role, but if you dont want to specify a route to one of the roles dont worry we set <front> as default value.
 -save and have fun!!.
